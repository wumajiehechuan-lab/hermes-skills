# 输出格式模板

> 单帧、序列、资产序列、视频提示词、End Frame 的输出格式。Phase 5 按需加载。

## 模式一：单张关键帧提示词

```
### 关键帧：[标题]
**用途**：[首帧/尾帧/转场帧/角色设定/环境设定]
**画面比例**：[16:9 / 9:16 / 1:1]

**Nano Banana Pro 提示词**：
[直接可复制使用的完整提示词]

**参考图需求**：[有→"已提供，附图生成"；无→"无参考图，纯文字描述生成"]

**生成建议**：[针对这张图的生成注意事项]
```

## 模式二：关键帧序列提示词

```
## 关键帧序列：[项目名称]

### 生成顺序与依赖关系
| 序号 | 关键帧 | 类型 | 依赖 | 优先级 |
|------|--------|------|------|--------|
| 1 | ... | 角色设定 | 无 | ★★★★★ |
| 2 | ... | 环境概念 | 无 | ★★★★★ |
| 3 | ... | 叙事构图 | 依赖#1、#2 | ★★★★☆ |

### 一致性锚点
- **角色一致性**：[如何维护角色一致性]
- **环境一致性**：[如何维护环境风格统一]
- **色调一致性**：[如何维护整组图的色彩基调]

---

### 帧1：[标题]
（同模式一结构）

### 帧2：[标题]
...
```

## 序列生成顺序原则

1. **先角色后场景**：角色设定图优先生成
2. **先环境后构图**：独立环境概念图优先
3. **先全局后细节**：大远景/全景优先确定氛围
4. **关键帧 > 过渡帧**：高潮画面、首帧、定格帧优先

## 模式三：TVC 产品资产序列

```
## 产品资产序列：[产品名称]

### 产品资产清单
| 序号 | 资产类型 | 用途 | 优先级 |
|------|---------|------|--------|
| 1 | 产品 Hero Shot | 锁定产品主视觉 | ★★★★★ |
| 2 | 产品材质微距 | 锁定材质质感 | ★★★★☆ |
| 3 | 品牌世界环境 | 锁定使用场景氛围 | ★★★★☆ |
| 4 | 产品交互特写 | 锁定使用动作/手势 | ★★★☆☆ |

### 产品一致性锚点
- **产品外观**：[产品的形状、比例、标志性设计特征的精确描述]
- **材质/光影**：[产品表面材质类型、反射特性、推荐打光方式]
- **品牌色**：[主色/辅色/点缀色的具体色值或描述性定义]
```

## 模式四：TVC Multi-Phase 视频提示词

```
## TVC 视频提示词：[项目名称] · Segment [N]（[时段]）

**对应多宫格**：Grid [N]
**世界类型**：产品世界 / 品牌世界 / 交叉
**时长**：[X]s

**Multi-Phase 视频提示词**：
风格：[整体视觉风格描述]
Phase 1 (0-Xs): [运动/动作/画面内容描述]
Phase 2 (X-Ys): [运动/动作/画面内容描述]
...
光影要求：[该段落的光影调性]

**首帧关键帧引用**：[对应的关键帧资产编号或名称]

**衔接备注**：
- 与上一段的衔接：[尾帧→首帧的衔接方式]
- 与下一段的衔接：[本段尾帧如何为下一段铺垫]
```

## 模式五：End Frame 专用输出

```
### End Frame：[产品名]
**用途**：TVC 收束定格
**画面比例**：16:9

**Nano Banana Pro 提示词**：
[直接可复制使用的完整提示词]

**文字叠加区域**：[Logo 和 Slogan 预留位置描述]

**参考图需求**：[是否需要品牌 VI 参考图]

**后期叠加指引**：
- **Logo 位置**：[具体区域描述]
- **Slogan 位置**：[具体区域描述]
- **推荐字体风格**：[与品牌调性匹配]
- **推荐字色**：[基于画面底色的文字颜色建议]
```

## TVC 序列生成顺序原则

1. **先产品后场景**：产品 Hero Shot 最先生成
2. **先产品后品牌世界**：产品资产确认后再生成品牌世界
3. **先多宫格后视频提示词**：关键帧锁定后再生视频提示词
4. **End Frame 最后**：所有视觉锁定后最后生成
