# 三类资产生成标准

> 产品图、角色资产图、场景图的生成标准 + 完整 Phase 4 操作流程。Phase 4 加载。

## 交互策略（完整流程）

1. 根据分镜脚本自动推导资产清单
2. 产品参考图已在 Phase 1 确认，此处不再追问产品层面
3. **一次性询问其他资产的参考图**：模特是否有参考照片？场景是否有参考图？——一次询问，不逐项追问，不索要图片、不等待用户发图
4. 按对应路径直接生成 prompt：
   - **默认路径（有参考图）** → prompt 直接引用参考图，**不描述外观细节**
   - **例外路径（无参考图）** → prompt 用文字精确描述外观，纯文生图
5. 然后问用户"满意吗？有什么要调整的？"

## 一、产品图

**目的**：锁定产品外观、材质、配色、比例。

**TVC 默认：产品多视图**。一张图包含多角度全身 + 关键细节特写。

### 两条路径

| 路径 | prompt 写法 |
|------|-----------|
| **默认（有参考图）** | `参考图片，为这个产品生成多视图，白底影棚，统一侧光+轮廓光...` |
| **例外（无参考图）** | 必须用文字精确描述产品外观（材质、配色、形状、设计特征）|

### 产品多视图模板

默认路径（有参考图）：
```
参考图片，为这个产品生成多视图。白底影棚，统一侧光+轮廓光。
包含以下视角：3/4经典角度全身、正面全身、侧面全身、背面全身，
以及2-3个关键细节微距特写（如瓶盖/按键/材质纹理/接口等产品最具辨识度的局部）。
保持所有视角的光影方向和色温一致。
```

例外路径（无参考图）：
```
白底影棚，[产品完整外观描述]的多视图。统一侧光+轮廓光。
包含以下视角：3/4经典角度全身、正面全身、侧面全身、背面全身，
以及[具体细节部位1]微距特写和[具体细节部位2]微距特写。
保持所有视角的光影方向和色温一致。
```

**核心要求**（共用）：白底/纯色影棚，光影至少双光源，例外路径材质必须精写。

## 二、角色资产图

**目的**：锁定人物体态、服装、配饰、气质。

### 格式决策

| 面部是否出镜 | 资产图格式 | 说明 |
|------------|----------|------|
| **含面部** | **完整三视图**：面部特写 + 全身正面 + 全身背面 | 标准格式 |
| **无面部/遮挡** | **造型展示图**：全身正面 + 全身侧面 + 全身背面 + 关键配饰特写 | 重点锁定服装和配饰 |
| 仅 1-2 格远景剪影 | **不做资产图**，用出镜者标准描述代替 | 见一致性部分 |

**核心要求**：
- 资产图永远是全身（禁止半身/局部）
- 白底，统一光影
- 服装与品牌调性统一
- 表情保持中性自然

### 穿戴产品的角色资产必须引用产品图

当产品穿在身上/戴在手上时，角色资产图提示词**每格都必须重复产品引用**：
`脚穿图1的跑鞋` / `手腕佩戴图1的手表` / `身穿图1的外套`

单次提及的权重不足以让 AI 在所有视角中正确渲染产品。

## 三、场景图

**目的**：锁定品牌世界的视觉氛围——空间、光影、色调、时间。

### 两条路径

| 路径 | prompt 写法 |
|------|-----------|
| **有参考图** | `参考图片，基于图中场景氛围生成 16:9 宽幅空镜，无人物无产品，保持相同色调和光影基调` |
| **无参考图** | 必须用文字精确描述空间、色调、光影、时间 |

**核心要求**：无人物无产品（只锁定空间），时间天气必须明确，色调是核心交付，宽幅 16:9。

### 做几张？

分镜中每一个**视觉上明显不同的空间**做一张，同一空间不同角落不分开做。

## 一致性维护

### 产品标准描述（必须建立）

```
[产品名]，[核心材质] + [配色]，[关键设计特征1]，[关键设计特征2]。
```

### 出镜者标准描述（有人出镜但不做角色资产时必须建立）

```
[体态] + [服装款式+颜色+材质] + [鞋/配饰]
```

服装款式和颜色跨格不可变化——不可在 G1 写"黑色短裤"而在 G2 写"黑色长裤"。

### 跨资产协调

- 产品色调和场景色调要协调
- 人物服装色与品牌色/场景色有呼应
- 所有资产统一使用同一画风方向
