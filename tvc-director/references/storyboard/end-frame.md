# End Frame 系统

> End Frame 标准构成、提示词模板、构图策略、后期叠加指引。Phase 5 加载。

## End Frame 标准构成

- 产品居中或偏置（视品牌规范而定）
- 品牌 Logo（通常在产品上方或下方）
- Slogan/Tagline（简短文字）
- 干净背景（纯色/微妙渐变/品牌色）

## End Frame 提示词模板

```
[画质锚定]，[产品描述][居中/偏置]静置于[背景描述]中央。[光影设计]。
产品下方/上方留出空间用于放置品牌标识。整体画面干净、高级、克制。[画风锚定]。
```

注意：Nano Banana Pro 不擅长精确文字渲染——Logo 和 Slogan 文字在后期叠加，提示词中只需预留空间。

## 多宫格 End Frame 格模板

```
[格号]. [景别]·[视角]：[背景描述]，[产品名称]以[角度/姿态]静置于画面[位置]，
[产品最终状态描述]。[简洁光影]。
[留白区域位置]留出画面[百分比]的干净空间，背景渐变均匀无杂物。
```

## 构图策略

### 产品居中（Hero Pose）
产品处于画面正中央，Logo 在产品上方或下方居中。适合需要最大产品冲击力的品牌。

### 产品偏置留白
产品位于画面左侧或左下三分之一处，右上方留出 30%-40% 空间给 Logo 和 Slogan。更现代、更杂志化。

## End Frame 三法则

1. **3 秒可读**：所有信息必须在 3 秒内被观众读取完毕
2. **3 个元素以内**：产品 + Logo + Slogan 最多三个信息层级
3. **3 层视觉层级**：明确的主-次-辅视觉优先级

## 文字/Logo 安全区域

| 区域 | 定义 | 用途 |
|------|------|------|
| **画面安全区** | 边缘内缩 5% | 所有重要视觉元素 |
| **标题安全区** | 边缘内缩 10% | 所有文字信息 |
| **上方 1/5** | 顶部 20% | Tagline、标题 |
| **下方 1/5** | 底部 20% | 产品信息、法律声明 |
| **中央 60%** | 中心区域 | 核心视觉内容，避免被文字遮挡 |
