# 品牌世界设计

> 双世界法则、三种品类路径、产品植入策略、出镜策略、视听转化法则。Phase 2 加载。

## 双世界法则

不让产品干瘪地存在于真空中。思考它在两个世界中的存在方式：

- **产品世界 (Product World)**：绝对的主场。微距、精密感、材质肌理、局部特写、光影流转。
- **品牌世界 (Brand World)**：产品所处的情感与文化语境。载体因品类而异。
- **交互枢纽 (Match Cut/Transition)**：通过形状匹配、动作连贯或色彩渐变，将两个世界的镜头无缝衔接。

## 三种品类路径

拿到 brief 后，先判断产品属于哪条路径：

### 功能型品牌世界（科技/工具/户外装备）
载体是**环境**。产品在极限/专业使用场景中出场。
- 运动相机 → 跳伞、潜水、滑雪、攀岩
- 专业工具 → 工地、实验室、手术室
- 产品世界的表达：零件拆解、功能激活、屏幕点亮、数据可视化

### 身份型品牌世界（时尚/珠宝/奢侈品/潮牌）
载体是**人**。人的气质、穿搭、姿态就是品牌世界。
- 先锋银饰 → 穿极简黑衣的人 + 建筑美学空间
- 奢侈珠宝 → 高定礼服 + 仪式感 + 优雅姿态
- 产品世界的表达：工艺溯源、佩戴流转、肌肤与材质的触感对比

### 感官型品牌世界（食饮/护肤/香氛）
载体是**感官体验**。原料质感、制作仪式、使用瞬间的感官细节。
- 精品咖啡 → 烘焙过程、萃取瞬间、蒸汽升腾
- 护肤品 → 晨光仪式、水珠滑落、质地微距
- 产品世界的表达：原料溯源、制作过程可视化、倾倒/涂抹/品尝的感官瞬间

## 产品植入策略

双世界法则定义了世界存在，但没有回答它们怎么共处。拿到 brief 后先判断品类路径，然后选择植入策略：

### 产品电影化拆解
产品是唯一主角，纯影棚全片。品牌世界可以不出现。
- 适合有内部结构可展示的产品：耳机、相机、手机、机械表芯
- 视觉逻辑：暗调揭示 → 悬浮拆解 → 功能激活 → 微距潜入 → 品牌定格
- 产品世界占比 100%

### 品牌世界穿梭
品牌世界和产品世界轮流出场，用 Match Cut 衔接。切换发生在 Phase 之间。
- 适合产品功能与极限场景强绑定的品类
- 视觉逻辑：品牌世界建立能量 → Match Cut → 产品世界展示 → Match Cut 回品牌世界 → 产品世界收束

### 生活方式短片
产品始终待在品牌世界中，不跳出去做影棚特写。通过运镜在场景内自然突出产品。
- 适合穿戴型产品：跑鞋、手表、眼镜、首饰、手袋
- 视觉逻辑：品牌世界叙事为主 → 产品通过运镜被"看见"而非被"展示" → 片尾 Hero Shot

### 品类直觉（默认倾向，可打破）
- 有内部结构可拆的科技产品 → 电影化拆解
- 产品功能绑定极限/专业场景 → 品牌世界穿梭
- 穿戴型 / 生活方式 → 生活方式短片

## 出镜策略

### 出镜决策树

| 品类路径 | 默认策略 |
|---------|---------|
| 身份型（珠宝/时尚/奢侈品）| 默认需要人物。产品佩戴在身体上，造型方向：服装风格、配饰、肤质、气质关键词 |
| 功能型（科技/工具/户外）| 人物可选。纯产品电影化拆解可无人；品牌世界穿梭需人物在使用场景中出现 |
| 感官型（食饮/护肤/香氛）| 视创意方向而定。使用仪式→手部/面部局部；原料溯源→可无人 |

### Seedance 多宫格人物约束

Seedance 多宫格（3x3 分镜）**不支持完整人脸**。人物出镜方式限于：
- 手部 / 手腕特写
- 下半脸（嘴唇、下颌线）
- 身体局部特写（肩颈、锁骨、腰线）
- 全身远景 / 背影 / 剪影
- 戴口罩 / 潜水面具 / 头盔等遮挡面部

**Nano Banana Pro 资产图不受此限制**——角色三视图可包含完整人脸，限制仅在多宫格分镜阶段生效。

### 造型方向

- **服装风格**：极简/高定/街头/工装/运动
- **色彩关系**：与产品色的呼应或对比
- **配饰逻辑**：除主推产品外是否有其他配饰？保持克制
- **肤质/妆发方向**：干净/自然/戏剧化/无妆感

## 视听转化法则

- **色彩与光影**：主色调是什么？品牌色如何自然植入？
- **视觉隐喻**：用画面代替说教。"新鲜"靠"清晨嫩芽滴落的露珠"；"静音"靠"一根针掉落的涟漪"
- **环境音场**：设计画面时同步思考声音
