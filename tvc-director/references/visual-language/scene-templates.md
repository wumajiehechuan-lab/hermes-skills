# TVC 场景类型模板

> 12 种 TVC 专用场景类型的提示词模板。Phase 5 分镜时按需加载对应场景类型。

## 场景类型总览

| 类别 | 场景类型 | 核心功能 |
|------|---------|---------|
| 产品世界 | 产品 Hero Shot | 主视觉定调 |
| 产品世界 | 产品电影化拆解帧 | 技术实力可视化 |
| 产品世界 | 功能可视化帧 | UI/数据/传感器呈现 |
| 产品世界 | 材质微距帧 | 工艺细节放大 |
| 产品世界 | Pack Shot | 包装展示 |
| 产品世界 | End Frame | 收束定格 |
| 品牌世界 | 极限运动场景 | 极端环境说服力 |
| 品牌世界 | 生活方式场景 | 日常使用共鸣 |
| 品牌世界 | 情感场景 | 人物与产品情感连接 |
| 品牌世界 | 环境氛围空镜 | 品牌世界观建立 |
| 交叉衔接 | 匹配剪辑过渡帧 | 世界间视觉桥梁 |
| 交叉衔接 | 产品 Reveal 过渡 | 场景→特写揭示转场 |

## 第一类：产品世界镜头

### 1. 产品 Hero Shot

```
[画质锚定]，[产品描述]静置于[背景]中央。[光影设计——侧光/轮廓光/顶光]。[材质细节]。[构图——产品占比/留白]。[画风锚定]。
```

**关键要点**：背景克制（纯色/极简影棚），光影至少双光源，材质必须精写，产品占比 40%-60% 最佳。

### 2. 产品电影化拆解帧

```
[画质锚定]，[产品名]的[组件描述]悬浮拆解状态。[各组件的空间位置关系]。[核心传感器/芯片发光效果]。[影棚侧光]。[画风锚定]。
```

**关键要点**：悬浮间距需描述空间层次关系，加入微弱自发光效果，3-5 个组件层级最佳。

### 3. 功能可视化帧

```
[画质锚定]，[产品]的屏幕/界面展示。[屏幕内容详细描述——UI元素/数据]。[屏幕发光对周围环境的影响]。[产品本体在画面中的位置]。[光影——以屏幕自发光为主光源]。[画风锚定]。
```

**关键要点**：屏幕内容用几何形状描述 UI 布局，避免要求精确文字；暗环境增强效果。

### 4. 材质微距帧

```
[画质锚定]，微距特写[产品部位]的[材质类型]表面。[微观物理细节——纹理/颗粒/反射]。[微距光影——侧光刻画纹理]。[极浅景深，焦点位置]。[画风锚定]。
```

**关键要点**：低角度侧光是微距之王，加入灰尘颗粒等微观参照物暗示放大倍率。

### 5. Pack Shot

```
[画质锚定]，[产品包装盒描述]正面朝向镜头，[摆放状态]。[包装盒旁边的产品本体]。[光影——柔和影棚光]。[构图]。[画风锚定]。
```

### 6. End Frame

```
[画质锚定]，[产品描述]以[角度/姿态]静置于[背景]中。[光影——干净、高级]。[构图——产品偏置，留白用于文字叠加]。[画风锚定]。
```

**关键要点**：为文字预留至少 30%-40% 干净区域，产品姿态稳定不动感。

## 第二类：品牌世界镜头

### 7. 极限运动场景

```
[画质锚定]，[运动场景环境描述]。[运动员动作/姿态/装备]。[产品在运动员身上的位置和状态]。[运动产生的物理效果——飞溅/气流/雪花]。[光影——自然光]。[构图——动感]。[画风锚定]。
```

### 8. 生活方式场景

```
[画质锚定]，[日常场景环境——时间/地点/氛围]。[人物衣着/状态/情绪]。[人物与产品的交互动作]。[环境细节]。[光影——自然光/窗光/暖调]。[构图]。[画风锚定]。
```

**关键要点**："不经意"感是关键，环境道具需有真实使用痕迹感。

### 9. 情感场景

```
[画质锚定]，[情感场景环境]。[人物关键情绪表情和肢体语言]。[产品在情感交互中的角色]。[光影——配合情绪的色温和明暗]。[构图——特写突出表情]。[画风锚定]。
```

### 10. 环境氛围空镜

```
[画质锚定]，[空间类型和规模]。[环境物理细节]。[大气效果——雾气/光线/粒子]。[光影设计]。[广角/俯拍]。[画风锚定]。
```

## 第三类：交叉衔接帧

### 11. 匹配剪辑过渡帧

设计 Match Cut 中的"桥梁帧"，在 A 尾帧与 B 首帧之间建立视觉连续性。

### 12. 产品 Reveal 过渡

从品牌世界使用场景平滑过渡到产品特写。三阶段：
- **阶段 A**（品牌世界→产品初现）：使用场景中景，人物动作使产品进入视线
- **阶段 B**（过渡帧→产品放大）：近景特写，产品占画面主体，环境虚化
- **阶段 C**（产品世界→Hero Shot）：使用产品 Hero Shot 模板
