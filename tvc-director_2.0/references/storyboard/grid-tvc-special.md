# TVC 多宫格特殊写法

> TVC 专用：产品世界/品牌世界/交叉 grid 写法、产品出镜率铁律、End Frame 格。Phase 5 加载。

## 产品世界 Grid

**定位**：低密度，每格以 `[景别·视角]：` 精确开头。

**写法要点**：
- 产品角度按逻辑编排：从整体到局部，从外观到内部，从静态到功能态
- 光影系统全局统一，仅色温/强度微变
- 背景极度克制：纯色渐变或纯黑
- 材质关键词保持一致

## 品牌世界 Grid

**定位**：低密度（TVC 默认），角色在品牌世界场景中使用产品。

**产品可见硬规则**（不可违反）：
- 产品在每格中必须可见——品牌世界不等于"没有产品的风景片"
- 产品占画面 10%-25%，自然融入但必须可辨识
- 产品在画面中的位置必须明确描述
- 产品屏幕/指示灯的发光颜色要写

## 交叉 Grid

**定位**：同一张 Grid 中混合品牌世界与产品世界面板。

**写法要点**：
- 奇数格（1/3/5/7）为品牌世界场景
- 偶数格（2/4/6/8）为产品世界特写——低密度，精确摄影指令
- 第 9 格固定为 End Frame
- 相邻格之间设计视觉衔接（形状匹配/色彩匹配/纹理匹配）

## 产品出镜率铁律

TVC 是产品广告，不是风景片——产品必须是每一帧的主角或重要配角。

- **全片产品可见格占比不低于 70%**
- **单张 Grid 无产品格不超过 2 格**
- **禁止连续 3 格以上无产品**
- **品牌世界格中产品也必须可见**：占画面 10%-25%

### 产品出镜验证（输出规划表后、生成提示词前必须执行）

扫一遍规划的全部 grid，在规划表的"产品出镜"列中标注每张 grid 的产品可见格数。违反铁律则必须调整分镜设计。

## End Frame 格

**总原则**：布局依品牌规范——产品居中（Hero Pose）或产品偏置留白均为有效方案。

**核心规则**：
1. 构图必须为后期文字预留空间（约 30%-40% 干净区）
2. 产品状态必须是"最终态"：屏幕亮起、外观完整、姿态稳定沉静
3. 光影简洁高级，不写 Logo/Slogan 文字内容，仅描述留白区域

**模板**：
```
[格号]. [景别]·[视角]：[背景描述]，[产品名称]以[角度/姿态]静置于画面[位置]，
[产品最终状态描述]。[简洁光影]。
[留白区域位置]留出画面[百分比]的干净空间，背景渐变均匀无杂物。
```

## 产品一致性维护

### 规则一：锁定产品描述模板
```
[产品名]，[核心材质A] + [核心材质B] 机身，
[颜色/配色方案]，[关键设计特征1]，[关键设计特征2]，
[尺寸/比例特征]。
```

### 规则二：统一光影关键词
同一产品在不同 Grid 中应使用相同的光影词汇集。

### 规则三：角度描述标准化
跨网格同一角度必须使用完全相同描述。

### 规则四：品牌世界产品最低描述
即使产品只占 10%，仍需包含：安装位置、材质关键词、功能状态提示、产品配色。

### 规则五：产品参考图全局引用
所有网格引用同一张产品资产图。

### 规则六：出镜者服装一致性
跨网格服装款式、颜色、材质三要素不可变化。复用前期筹备建立的出镜者标准描述。

### 规则七：禁止比喻修辞
AI 图片模型是字面理解的——"如雕塑"会生成石膏材质，"如丝绸"会改变材质渲染。

| 错误写法 | 正确写法 |
|---------|---------|
| 路人定格如**雕塑** | 路人保持行走姿态静止不动，表情凝固，衣角悬停在运动状态 |
| 在纸张之间穿行如同**采摘花朵** | 从空中逐张摘取悬浮的纸张 |

原则：**描述你要什么，不要描述它像什么。**

### 规则八：多宫格只写状态，不写过程
多宫格是静态关键帧，每格只能描述一个时间点的画面状态。动词必须是静态的（悬浮、静止、停驻、凝固）。

| 错误写法（过程态）| 正确写法（结果态）|
|----------------|----------------|
| 咖啡杯**脱手飞出**，液体**开始溢出** | 咖啡杯悬停在半空，液体凝固为弧形 |
| 红酒**开始涌出**杯口 | 红酒液柱悬浮在倾斜酒杯与桌面之间 |
