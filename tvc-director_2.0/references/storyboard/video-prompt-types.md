# 三种 TVC 视频提示词类型

> 产品电影化拆解、品牌世界穿梭、生活方式短片的视频写法详解 + 节奏表。Phase 5 加载。

## 类型一：产品电影化拆解型

每个 Phase 对应一个产品功能/卖点。产品状态精确描述，运镜高度精确，光影随产品动态流转。

### 六阶段节奏模型

```
超慢揭示 → 悬浮拆解 → 精密合拢 → 功能激活 → 微距潜入 → 品牌定格
```

| 阶段 | 节奏 | 运镜 | 产品状态 |
|------|------|------|---------|
| **超慢揭示** | 极慢/静止 | 固定→微推 | 全黑中轮廓光逐步勾勒 |
| **悬浮拆解** | 慢→渐快 | 弧形环绕 | 组件分离，内部暴露 |
| **精密合拢** | 爆发 | 多角度快切 | 组件高速回弹归位 |
| **功能激活** | 中速脉冲 | 推近至特写 | 屏幕点亮/传感器发光 |
| **微距潜入** | 超慢 | 微距滑行 | 材质微观细节放大 |
| **品牌定格** | 静止 | 弹射拉远→定格 | 产品完美形态+Logo |

### 阶段间转场编排

- 超慢揭示→悬浮拆解：轮廓光照亮全貌后，组件"呼吸式"微振→分离
- 悬浮拆解→精密合拢：组件悬停 0.3s 静默→突然加速回弹
- 精密合拢→功能激活：合拢瞬间白光闪线→屏幕从该光点扩散亮起
- 功能激活→微距潜入：屏幕像素→镜头穿入→进入材质微观世界
- 微距潜入→品牌定格：材质纹理逐渐模糊→弹射拉远至全景

## 类型二：品牌世界穿梭型

每个 Phase 完整待在一个世界——世界切换发生在 Phase 之间，不是 Phase 内部。Phase 之间通过 Match Cut / 运动连接衔接。

### 30s TVC 双段节奏（品牌世界穿梭型）

```
═══ 第一段（0-15s）：品牌世界主导 ═══
Phase 1 (0-3s): 品牌世界开场——极限场景广角大全景
Phase 2 (3-6s): 产品初现——[Match Cut] 场景运动→产品旋转
Phase 3 (6-10s): 品牌世界升级——场景升级，产品在使用中自然出现
Phase 4 (10-15s): 功能揭示——[Match Cut] 场景动作→产品功能激活

═══ 第二段（15-30s）：产品主导 ═══
Phase 5 (15-19s): 承接 + 深度拆解
Phase 6 (19-23s): 精密合拢 + 爆发旋转
Phase 7 (23-27s): 微距潜入
Phase 8 (27-30s): 品牌定格
```

## 类型三：生活方式短片型

产品始终在品牌世界中（穿在身上/戴在手上），不跳出去做影棚特写。通过运镜手法在场景内自然突出产品。片尾集中做 Hero Shot 收束。

### 三种切换技法

**Match Cut 动势传递**：品牌世界的运动→在特定角度/速度瞬间切换至产品世界，延续动势。

**尺度穿越**：宏观场景→镜头急速俯冲向一个点→穿透尺度后变成产品屏幕上的像素。

**介质穿越**：镜头对准产品表面→推入穿越镀膜/透镜/传感器→穿出后是品牌世界的壮丽风景。

### 切换节奏

| 视频时长 | 建议切换次数 | 节奏策略 |
|---------|------------|---------|
| 10s | 1-2 次 | 品牌开场→产品收束 |
| 15s | 2-3 次 | 品牌→产品→品牌→定格 |
| 30s | 4-6 次 | 前半段品牌主导，后半段产品主导 |

- 每次切换后至少停留 2-3s
- 最后一次切换必须是切入产品世界

## 15s 紧凑节奏

| 阶段 | 时长 | 功能 |
|------|------|------|
| **闪现揭示** | 0-2s | 建立产品印象 |
| **核心卖点** | 2-6s | 集中展示 1-2 个功能 |
| **视觉高潮** | 6-10s | 最具冲击力的画面 |
| **情绪锚点** | 10-13s | 记忆点 |
| **品牌定格** | 13-15s | Logo+产品+Slogan |

**15s 铁律**：单焦点原则（只传递 1 个核心记忆点）、禁止空镜、转场即内容、最后 2s 必须是静态定格。
