# 画风方向选择指南

> A-E 五种画风方向。Phase 3 画风确认时加载，Agent 展示选项给用户选择。

## 五种画风方向

| 选项 | 说明 | 视觉效果 | 典型参照 |
|------|------|---------|---------|
| **A. 真人实拍/摄影级** | 像真实摄影照片 | 产品摄影质感、苹果广告 | 电影剧照、时装摄影 |
| **B. 真人电影剧照** | 介于真人和CG之间 | 影视级真实感 | 漫威电影、权力的游戏 |
| **C. 3A游戏CG** | 高品质游戏CG渲染 | 精致理想化美感 | 最终幻想CG、原神过场 |
| **D. 高精CG引擎级** | 全CG但追求接近真实 | "CG还是真的？"的模糊感 | 头号玩家、虚幻引擎5 Demo |
| **E. 特定美学风格** | 水墨、赛博朋克、动漫等 | 风格化艺术表达 | 根据具体风格而定 |

## 选择建议

| 产品类型 | 推荐画风 | 理由 |
|---------|---------|------|
| 电子产品（手机/耳机/相机）| C 或 D | CG可精细控制产品外观和材质 |
| 户外运动产品 | A 或 B | 真人实拍强化极限场景真实感 |
| 时尚/珠宝/奢侈品 | A 或 D | 真人强调质感，引擎级CG追求完美 |
| 食饮/护肤品 | A | 真实感传达"想吃/想用"的欲望 |
| 概念产品 | E 或 D | 可脱离现实约束 |

## 确认规则

**[硬约束]** 即使用户描述中看似已明确画风，也必须复述所理解的方向并请用户确认。在用户明确回复确认之前，不得生成任何提示词。

确认后，全套关键帧统一使用同一画风方向。
