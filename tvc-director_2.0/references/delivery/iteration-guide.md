# 迭代优化指南

> 迭代策略、常见失败模式、修正方案、微调技巧。Phase 6 加载。

## 迭代核心原则

### 1. 单变量原则

每次迭代只修改**一个变量**。同时改三处无法判断哪个修改起了作用。

### 2. 加减法原则

- **多了不想要的** → 减法：删除可能暗示该元素的词汇
- **少了想要的** → 加法：加入更具体的描述词
- **方向错了** → 替换：换用含义更精确的词

### 3. 位置权重原则

提示词**越靠前的词权重越高**。效果不够强→把对应描述移到更前面。

### 4. 避免"越改越差"

超过 3 次微调无效时，退一步分析根本原因。不要从一个极端摆到另一个极端。

## 常见失败模式与修正

### 模式1：画面太亮/不够暗

加入：环境极度昏暗 / 几乎全黑 / 大部分空间被深邃的阴影吞没。删除任何暗示额外光源的词。

### 模式2：出现不想要的物体

删除可能暗示该物体的描述。用更具体的空间描述替代模糊的空间类型词。

### 模式3：空间感不够

加入：天花板高耸入云（垂直维度）/ 空气中弥漫体积雾（大气透视）/ 极低的视角（极端透视）。

### 模式4：人物面部不好看/变形

简化面部描述，只保留 1-2 个特征。上传角色参考图。用侧颜/半剪影规避正脸。

### 模式5：色调不对

明确色调词（去饱和冷灰蓝 / 血橙深红 / 琥珀暖金）。确保光源颜色与目标色调一致。

### 模式6：构图散乱/主体不突出

加入明确构图法则。减少非核心元素。加入景深控制。

### 模式7：材质质感不够

金属加入"拉丝纹理/微氧化痕迹"。皮肤加入"毛孔/细微汗珠"。布料加入"编织纹理/毛边"。

### 模式8：画面太平/缺层次

加入前中远景分层。加入体积雾/尘埃颗粒。指定焦点位置。

### 模式9：CG感过重/不像真人

将所有CG方向锚定词替换为真人方向：`真人实拍，电影级摄影，真实皮肤毛孔质感，自然光照`。删除"大师级CG渲染"等CG词。

### 模式10：真人感过重/想要CG

将真人方向锚定词替换为CG方向：`3A大作3D游戏风格 / 大师级CG渲染 / 虚幻引擎5渲染`。

### 模式11：风格混乱/真人和CG摇摆

检查所有画风画质相关词，标记方向归属。确保全部指向同一方向。

## TVC 专属迭代重点

- **产品材质不对** → 调整材质描述词，参照 product-breakdown-camera.md 的材质微距速查表
- **产品光影太平** → 加强侧光/轮廓光描述
- **品牌世界不够极限** → 加强环境极端性描述
- **产品在场景中不够突出** → 调整产品描述的位置权重

## 微调技巧

### 加词的影响

| 加入的词 | 通常的影响 |
|---------|-----------|
| "极其"、"极度" | 增强紧跟其后的描述词效果 |
| "微弱的" | 减弱光源/效果的强度 |
| "几乎全黑" | 大幅压暗整体画面 |
| 具体数字（如"60米"）| 帮助AI建立尺度概念 |
| 画面占比（如"占画面70%"）| 精确控制元素大小 |

## 快速诊断清单

当生成结果不理想时，逐项排查：

- [ ] 画风方向对不对？
- [ ] 画质是否达标？
- [ ] 主体对不对？
- [ ] 空间感对不对？
- [ ] 太亮还是太暗？
- [ ] 色调对不对？
- [ ] 构图对不对？
- [ ] 有没有多余元素？
- [ ] 材质质感够不够？
- [ ] 空间层次够不够？

## 迭代工作流

```
1. 首次生成 → 2. 评估结果 → 3. 诊断问题（色调/光影/构图/主体？）
→ 4. 单变量修改 → 5. 二次生成 → 6. OK→下一个维度，不OK→换策略
→ 7. 超过3次无效→退一步分析根本原因
```
