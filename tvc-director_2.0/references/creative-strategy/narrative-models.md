# TVC 叙事模型

> A-H 八种 TVC 专用叙事模型 + 选择指南 + 60s 适配方案。Phase 2 创意提案时加载。

## 八种叙事模型

### Model A：直给型 (Direct Pitch)
- **结构**：产品登场 → 核心功能 → Hero Shot → End Frame
- **适用**：功能明确、无需铺垫的科技产品
- **节奏**：快，15-30s 最佳
- **产品世界占比**：90%+

### Model B：问题解决型 (Problem-Solution)
- **结构**：痛点 → 产品介入 → 问题解决 → Hero Shot
- **适用**：有明确痛点的功能性产品
- **节奏**：中速，30s 最佳
- **产品世界占比**：60%

### Model C：品牌世界穿梭型 (World-Hopping)
- **结构**：品牌世界建立 → Match Cut 到产品世界 → 回到品牌世界 → 产品世界收束
- **适用**：产品功能与极限/专业场景强绑定
- **节奏**：中快交替，30-60s 最佳
- **产品世界占比**：40-50%

### Model D：生活方式型 (Lifestyle)
- **结构**：日常生活 → 产品在其中 → 情感高点 → Hero Shot
- **适用**：穿戴型产品、生活方式品牌
- **节奏**：舒缓中速，30-60s 最佳
- **产品世界占比**：20%（产品始终在品牌世界中）

### Model E：工艺溯源型 (Craft Origin)
- **结构**：原材料 → 制作过程 → 成品亮相 → 使用场景
- **适用**：强调工艺/材质的品牌（珠宝、手表、奢侈品）
- **节奏**：慢，30-60s 最佳
- **产品世界占比**：50%

### Model F：对比型 (Before-After)
- **结构**：使用前状态 → 产品介入 → 使用后转变
- **适用**：功效型产品（护肤、清洁、工具）
- **节奏**：中快，15-30s 最佳
- **产品世界占比**：40%

### Model G：悬念揭示型 (Reveal)
- **结构**：局部特写/抽象画面 → 逐步揭示 → 全貌亮相
- **适用**：设计感强、外观惊艳的产品
- **节奏**：慢→爆发，15-30s
- **产品世界占比**：80%

### Model H：证言型 (Testimonial)
- **结构**：使用场景 → 产品效果 → 第三方认可 → Hero Shot
- **适用**：需要信任背书的品类
- **节奏**：中速，30s+
- **产品世界占比**：30%

## 模型选择速查

| 产品类型 | 首选模型 | 次选 |
|---------|---------|------|
| 科技硬件（手机/相机/耳机） | A 直给型 / G 悬念揭示 | C 品牌世界穿梭 |
| 户外装备 | C 品牌世界穿梭 | A 直给型 |
| 穿戴设备（手表/手环） | D 生活方式 | A 直给型 |
| 奢侈品/珠宝 | E 工艺溯源 | D 生活方式 |
| 功效护肤品 | F 对比型 | B 问题解决 |
| 汽车 | C 品牌世界穿梭 | G 悬念揭示 |
| 食饮 | D 生活方式 | E 工艺溯源 |

## 60s 适配方案

每个模型在 60s 时长下拆分为 4 个 15s 叙事单元：
- **0-15s (起)**：建立
- **15-30s (承)**：展开
- **30-45s (转)**：高潮
- **45-60s (合)**：收束 + End Frame
